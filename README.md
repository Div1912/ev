# EduVerify

**EduVerify** is a decentralized platform that leverages blockchain technology to issue, verify, and manage educational and professional credentials. It ensures tamper-proof, privacy-preserving verification while providing a digital reputation system and AI-powered resume builder.

- **Blockchain-Based Credentialing**  
  Issue and verify academic and professional credentials securely on a decentralized ledger.

- **Tamper-Proof Verification**  
  Credentials cannot be forged or altered, ensuring authenticity.

- **Privacy-Preserving**  
  User data is protected while still allowing verification by authorized entities.

- **Digital Reputation System**  
  Build a trusted digital identity and showcase verified skills and achievements.

- **AI Resume Builder**  
  Generate optimized resumes based on verified credentials and professional experience.
